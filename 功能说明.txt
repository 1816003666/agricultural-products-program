# 农产品电商网站功能文档
## 1. 项目概述
本项目是一个基于 Vue 3 + Vite 构建的农产品电商网站，旨在为用户提供新鲜、优质的农产品购买服务。网站分为前端和后端两部分，前端负责用户界面和交互，后端负责数据处理和业务逻辑。

## 2. 技术栈
- 前端 ：Vue 3、Vite、JavaScript
- 后端 ：（位于 backend 目录，具体技术栈未详细分析）
- API 调用 ：通过 services/api.js 实现
- 状态管理 ：使用 Vue 3 Composition API 的 ref 和 computed
- 路由管理 ：Vue Router
- 本地存储 ：localStorage
## 3. 核心功能模块
### 3.1 首页
功能描述 ：

- 展示网站导航栏、轮播图和各类农产品分类
- 支持商品搜索功能
- 显示购物车商品数量
- 按分类展示商品列表，包括：
  - 新鲜水果
  - 新鲜蔬菜
  - 肉禽蛋
  - 水产
  - 粮油
  - 干货
技术实现 ：

- 使用 ProductList 组件展示各类商品
- 集成 Carousel 组件实现轮播图
- 通过 API 或默认数据获取商品信息
- 支持搜索结果展示
### 3.2 产品详情
功能描述 ：

- 展示商品的详细信息，包括名称、价格、描述、图片等
- 支持将商品添加到购物车
- 支持关闭详情页面
技术实现 ：

- 使用 ProductDetail 组件实现
- 从父组件接收商品数据
- 调用购物车相关 API 添加商品
### 3.3 购物车
功能描述 ：

- 展示购物车中的商品列表
- 支持修改商品数量
- 支持删除购物车商品
- 支持结算功能
技术实现 ：

- 使用 Cart 组件实现
- 登录用户通过 API 获取购物车数据
- 未登录用户使用本地存储保存购物车数据
### 3.4 订单管理
功能描述 ：

- 查看订单列表
- 查看订单详情
- 跟踪订单状态
技术实现 ：

- OrderListPage 组件展示订单列表
- OrderDetailPage 组件展示订单详情
- 通过 API 获取订单数据
### 3.5 用户中心
功能描述 ：

- 查看和编辑个人资料
- 管理收货地址
- 查看个人订单历史
- 登录和注册功能
技术实现 ：

- ProfilePage 组件实现
- 通过 API 获取和更新用户信息
- 集成 LoginModal 和 RegisterModal 组件
### 3.6 管理员后台
功能描述 ：

- 仪表盘：查看网站概览数据
- 产品管理：查看、添加、编辑、删除产品
- 订单管理：查看和处理订单
- 用户管理：查看和管理用户
- 数据分析：查看销售和用户数据
技术实现 ：

- AdminPage 作为后台主页面
- 子路由实现各个管理模块
- 通过 API 实现数据管理功能
## 4. 页面结构

## 5. 数据管理
### 5.1 API 服务
- productAPI ：商品相关 API，包括获取商品列表、搜索商品等
- cartAPI ：购物车相关 API，包括添加商品、获取购物车数据等
- userAPI ：用户相关 API，包括登录、注册、获取用户信息等
- orderAPI ：订单相关 API，包括创建订单、获取订单列表等
### 5.2 本地存储
- token ：用户登录凭证
- cart ：未登录用户的购物车数据
## 6. 响应式设计
- 适配不同屏幕尺寸，包括桌面端、平板和移动设备
- 使用媒体查询调整布局和样式
- 移动端友好的交互设计
## 7. 特色功能
1. 分类浏览 ：按农产品类别（水果、蔬菜、肉禽蛋、水产、粮油、干货）浏览商品
2. 搜索功能 ：支持商品关键词搜索
3. 购物车管理 ：支持登录用户和未登录用户的购物车功能
4. 用户中心 ：个人资料管理和订单历史查看
5. 管理员后台 ：完整的后台管理功能，包括产品、订单、用户管理和数据分析
6. 轮播图展示 ：首页轮播图展示推荐商品或促销活动
## 8. 流程说明
### 8.1 商品浏览流程
1. 用户访问首页，查看各类商品分类
2. 点击商品分类，浏览该分类下的商品
3. 点击商品卡片，查看商品详情
4. 在商品详情页，可选择数量并添加到购物车
### 8.2 购物流程
1. 用户将商品添加到购物车
2. 点击购物车图标，查看购物车内容
3. 修改商品数量或删除商品
4. 点击结算按钮，进入订单确认页面
5. 填写收货地址和支付方式
6. 提交订单，完成购买
### 8.3 管理员操作流程
1. 管理员登录后台
2. 在仪表盘查看网站概览数据
3. 进入产品管理页面，添加、编辑或删除产品
4. 进入订单管理页面，处理用户订单
5. 进入用户管理页面，管理用户信息
6. 进入数据分析页面，查看销售和用户数据
## 9. 总结
本农产品电商网站提供了完整的电商功能，包括商品浏览、搜索、购物车、订单管理、用户中心和管理员后台。网站采用现代化的前端技术栈，响应式设计，为用户提供良好的购物体验。同时，管理员后台功能完善，方便网站运营者管理商品、订单和用户数据。

网站的特色在于专注于农产品领域，提供了丰富的商品分类，满足用户对新鲜农产品的需求。通过本地存储和 API 调用相结合的方式，实现了流畅的用户体验，无论是登录用户还是未登录用户都能正常使用购物车功能。
